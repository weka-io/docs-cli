```
Usage:
smb share [--HOST HOST] [--PORT PORT] [--json]

Description:
    List all shares exposed via samba

Subcommands:
   add      Add a new share to be exposed by samba
   remove   Remove a share exposed by samba
   chown    Chown a share to a specific user

Options:
   -H, --HOST   Specify the host. Alternatively, use the WEKA_HOST env variable
   -P, --PORT   Specify the port. Alternatively, use the WEKA_PORT env variable
   -J, --json   Format output as JSON
```
