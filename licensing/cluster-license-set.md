```
Usage:
cluster license set <license> [--HOST HOST] [--PORT PORT] [--json] [--NO-HUMAN]

Description:
    Set the cluster license

Arguments:
   license   The new license to set to the system
Options:
   -H, --HOST       Specify the host. Alternatively, use the WEKA_HOST env variable
   -P, --PORT       Specify the port. Alternatively, use the WEKA_PORT env variable
   -J, --json       Format output as JSON
   -N, --NO-HUMAN   Print sizes in Bytes. When not set, sizes are printed in human readable format, e.g 1KiB 234MiB
                    2GiB.
```
