```
Usage:
cloud 

Description:
    Cloud commands. List the cluster's cloud status, if no subcommand supplied.

Subcommands:
   status        Show cloud connectivity status
   enable        Turn cloud features on
   disable       Turn cloud features off
   update        Update cloud settings
   creds         Commands that manage cloud's credentials.
   upload-rate   Get the cloud upload rate
   ui            Get the GUI service base URL

```
